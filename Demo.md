## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Documentation

```
Swagger Documentation can be found at http://localhost:3000/api-docs#/ for the local environment.
```
